# computer-file-browser
一款可以浏览电脑文件的android app 目前只可以查看，下载 文件。内置ijkplayer，支持视屏播放（列表，小窗口，全屏），图片展示，滑动删除，断点下载。需要在局域网下使用，服务器也已上传（目前只有tomcat版）
